@nodejs

Nodeのもっとも価値ある特徴は、フレンドリーで 色とりどりな開発者のコミュニティです。このグループが集まる場所がインターネット上にたくさんあります。このページは最高のフォーラムにハイライトを当てます。

ドキュメント

公式 API ドキュメント は node の API を詳述します。

docs.nodejitsu.com は、多くの人が一般的に遭遇する問題に答えてくれます。

How To Node には有用なチュートリアルが増え続けています。

Stack Overflow の node.js タグ には毎日新しい情報が集まっています。

GitHub

全ての開発は http://github.com/joyent/node で行われます。 コミットメッセージに対するコメントは、 しばしばコア開発における熱い議論の元となります。

GitHub Node wiki には、初心者に有益なリンクの全てがあります。 Node を使ったプロジェクト、アプリケーション、企業 や、その多くが npm レジストリ に公開されている とても長い Node モジュールの一覧 を見逃さないでください。

メーリングリスト

ユーザメーリングリスト は、Node に関するアナウンス、議論、バトルに使われます。 内部的な開発メーリングリスト は、内部的な設計や新機能の提案を議論するために使われます。

Periodicals

Planet Node は、 Node 開発者ブログのアグリゲーターです。 NodeUp は、 コミュニティの最新ニュースをカバーするポッドキャストです。

カンファレンス

NodeConf カンファレンスは米国における主要なイベントです; それは Mikeal Rogers によって運営されています。

東京Node学園祭 は Node.js 日本ユーザグループ によって運営されています。

ドイツ、ケルンでの NodeCamp.de は Rails Love によって運営されています。

Italian Node.js Conference も存在します。

サンフランシスコでの Node Summit カンファレンスは、大企業における Node の適用に フォーカスしています。

JSConf は JavaScript の主要なカンファレンスを主催します。

ローカライゼーションサイト

nodejs.org は他言語への翻訳を持ちません。 しかし、様々な言語によるメーリングリストや翻訳された Web サイトを含むコミュニティサイトが存在します。

nodejs.ru ロシア語によるブログ
nodejs.ir ペルシャ語によるイランのグループ
nodejs.jp 日本ユーザグループ
CNodeJS.org 中国語のコミュニティ
nodejs.co.il イスラエルの Wiki
HKNoJ 香港のコミュニティ
nodejs.tw 台湾のコミュニティ

IRC

Node 開発に関するリアルタイムチャットは、 irc.freenode.net の #node.js チャネルで行われています。 IRC クライアント または Web ブラウザで freenode's WebChat にアクセスしてください。 見逃した人のために、Felix Geisendörfer が ログ を保存しています。
