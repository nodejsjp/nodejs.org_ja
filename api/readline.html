Readline

このモジュールはストリーム (たとえば標準入力) を行ごとに読み込むことを可能にします。

このモジュールを一度起動すると、このインタフェースと (標準入力などの) ストリームをクローズするまで node プログラムは終了しないことに注意してください。 きれいに終了する方法を以下に示します:

<pre> var readline = require('readline');

var i = readline.createInterface(process.sdtin, process.stdout, null); i.question("What do you think of node.js?", function(answer) { //TODO: Log the answer in a database console.log("Thank you for your valuable feedback."); i.close(); //These two lines together allow the program to process.stdin.destroy(); //terminate. Without them, it would run forever. }); </pre>
