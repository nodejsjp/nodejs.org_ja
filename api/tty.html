TTY

require('tty') でこのモジュールにアクセスします。

例:

var tty = require('tty');
process.stdin.resume();
tty.setRawMode(true);
process.stdin.on('keypress', function(char, key) {
  if (key && key.ctrl && key.name == 'c') {
    console.log('graceful exit');
    process.exit()
  }
});

tty.isatty(fd) #

fd が端末に関連づけられているかどうかを true または false で返します。

tty.setRawMode(mode) #

mode は true または false です。 これは現在のプロセスの標準入力ファイル記述子をローデバイスまたはデフォルトに振る舞うよう設定します。

tty.setWindowSize(fd, row, col) #

この関数は v0.6.0 で削除されました。

tty.getWindowSize(fd) #

この関数は v0.6.0 で削除されました。 代わりに process.stdout.getWindowSize() を使用してください。
