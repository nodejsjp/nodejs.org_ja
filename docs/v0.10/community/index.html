@nodejs

Nodeのもっとも価値ある特徴は、フレンドリーで 色とりどりな開発者のコミュニティです。 このグループが集まる場所がインターネット上にたくさんあります。 このページは最高のフォーラムにハイライトを当てます。

ドキュメント

公式 API ドキュメント は Node の API を詳述します。

NodeSchool.io は、対話的な コマンドラインのゲームを通じて Node.js のコンセプトを 教えてくれます。

Stack Overflow の node.js タグ には毎日新しい情報が集まっています。

docs.nodejitsu.com は、多くの人が一般的に遭遇する問題に答えてくれます。

How To Node には有用なチュートリアルが増え続けています。

GitHub

全ての開発は http://github.com/joyent/node で行われます。 これは議論が発展する中心的な場所です。

もしあなたが Node.js に貢献することに興味があるなら、最初に 貢献 ガイド を最初に読んでください。

議論

ユーザ メーリングリスト は、Node に関するアナウンスや議論に 使われます。

The github 課題一覧は、Node.js コアの機能に関して議論する場所です。

多くの Node ユーザ が Twitter でも同様に 活動しています。

定期刊行物

Node.js Weekly は、 Node.js コミュニティからの最新のイベントやニュースを集めた メーリングリストです。

Planet Node は、 Node 開発者ブログのアグリゲーターです。

NodeUp は、 コミュニティの最新ニュースをカバーするポッドキャストです。

NodeJS Reactions は、アニメーション GIF で表現された Node.js に関する体験を 集めています。

イベント

Node Meatspace は、 様々な場所でこれから行われる会合やイベントを掲載する Web ページです。

NodeConf カンファレンスは米国における主要なイベントです; それは Mikeal Rogers によって運営されています。

NodeConf EU は、 Cian O'Maiden によって運営されているヨーロッパのカンファレンスです。

東京Node学園祭 は Node.js 日本ユーザグループ によって運営されています。

ドイツ、ケルンでの NodeCamp.de は Rails Love によって運営されています。

Italian Node.js Conference も存在します。

Node Summit は、 大企業における Node の適用にフォーカスした サンフランシスコのカンファレンスです。

JSConf は、 JavaScript の主要なカンファレンスを主催します。

IRC

Node 開発に関するリアルタイムチャットは、 irc.freenode.net の #node.js チャネルで行われています。IRC クライアント または Web ブラウザで freenode's WebChatにアクセスしてください。
